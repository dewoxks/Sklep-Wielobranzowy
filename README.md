Sklep Wielobranżowy
Opis projektu

System Sklep Wielobranżowy służy do zarządzania sprzedażą produktów w sklepie wielobranżowym. Aplikacja obsługuje klientów, zamówienia, płatności oraz kontrolę stanów magazynowych. System umożliwia również naliczanie cen z uwzględnieniem promocji i rabatów oraz generowanie raportów sprzedażowych.

Funkcjonalności

Klienci mogą składać zamówienia na dostępne produkty

Pracownicy mogą:

dodawać nowe produkty

aktualizować stany magazynowe

realizować zamówienia

System przechowuje historię zamówień i transakcji

Naliczanie cen z uwzględnieniem promocji i rabatów

Generowanie raportów sprzedaży i stanów magazynowych
